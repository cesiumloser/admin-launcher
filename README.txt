Admin Launcher
Утилита для быстрого запуска терминалов Windows с правами администратора.

Возможности
Быстрый запуск CMD с правами администратора

Быстрый запуск PowerShell с правами администратора

Простой интерфейс командной строки

Поддержка аргументов для автоматизации

Установка
Способ 1: Автоматическая установка (Windows)
Скачайте файлы проекта

Запустите install.bat

Способ 2: Ручная установка
bash
# Склонируйте репозиторий
git clone https://github.com/cesiumloser/admin-launcher.git
cd admin-launcher

# Установите глобально
npm install
npm link

==============================================================================================================

Admin Launcher
A utility for quickly launching Windows terminals with administrator privileges.

Features
Quick launch of CMD with administrator rights

Quick launch of PowerShell with administrator rights

Simple command-line interface

Support for command-line arguments for automation

Installation
Method 1: Automatic Installation (Windows)
Download the project files

Run install.bat

Method 2: Manual Installation
bash
# Clone the repository
git clone https://github.com/cesiumloser/admin-launcher.git
cd admin-launcher

# Install globally
npm install
npm link
